# Запуск на другой БД
Реализация для MySQL (не проверял, если честно, но должно работать)
1. Раскомментировать зависимость в pom
2. Создать БД
3. Проперти
 3.1. Либо в application.properties spring.profiles.default h2 поменять на mysql
 3.2. Либо запуск командой mvn spring-boot:run -Dspring-boot.run.profiles=mysql
4. Наслаждаемся

P.S. clean compile всему голова
